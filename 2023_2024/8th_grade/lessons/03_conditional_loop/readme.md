# לולאה מותנה (while)

## האתגר - משחק ניחושים

כללי המשחק:
- שחקן 1 בוחר מספר בין אחת ל1000
- לשחקן 2 יש 10 ניחושים למצוא את המספר
- בכל ניחוש שחקן 1 רק עונה האם הניחוש שווה למספר, גדול מהמספר או קטן מהמספר
